# 人体结构学 · 复习网页

期末周复习工具：点身体上的圆点，显示该结构名称；可自我测试。

## 怎么打开
双击 `index.html`（用 Chrome / Edge / Safari 打开即可，不需要联网）。

## 三种模式（左上角切换）
- **学习模式**：鼠标移到圆点上就显示名称。图上原有的印刷/手写标注也可见，适合先记忆。
- **测试模式**：图上原有文字被裁掉/遮住，只剩圆点。上方会提示"找出：XX"，你在图上点对应的圆点作答；点对变绿、点错闪红。也可以直接点任意圆点看答案。右侧清单显示"？"，点一项会闪烁对应圆点。
- **编辑模式**：校准/补充圆点用。
  - 单击图上空白处 → 新增一个圆点并输入名称
  - 拖动圆点 → 调整位置
  - 单击圆点 → 改名（内容留空即删除该点）
  - 改完点右下角 **导出数据**，会下载一个新的 `data.js`，用它覆盖项目里的旧 `data.js` 即可永久保存（否则修改只存在本机浏览器缓存里）。

## 已做成可点击的部位（顶部标签）
手、足、上肢、下肢、躯干、背部 —— 每个部位挑了一张最清楚的图。
**全部参考图** 标签里放了全部 23 张原图，点开可放大对照。

## 说明
- 圆点位置是我照着原图箭头**目测**放的，个别可能略有偏差；用**编辑模式**拖一下即可校准。
- "躯干""下肢"两张是打印纸+手写标注的照片，文字散布在身体上，测试模式没法完全遮干净——这两张更适合用学习模式看，或在编辑模式里微调后使用。
- 想把"全部参考图"里的其他图也做成可点击：在 `data.js` 的 `regions` 里仿照现有格式加一段，或直接在编辑模式里加点。

## 进度
测试进度和你的编辑都存在浏览器本地缓存。换浏览器/清缓存会丢失，所以重要的圆点修改记得用"导出数据"保存 `data.js`。
